# 19、SpringCloud 都用过哪些组件 介绍一下作用

(1) Nacos--作为注册中心和配置中心，实现服务注册发现和服务健康监测及配置信息统一管理

(2) Gateway--作为网关，作为分布式系统统一的出入口，进行服务路由，统一鉴权等

(3) OpenFeign--作为远程调用的客户端，实现服务之间的远程调用

(4) Sentinel--实现系统的熔断限流

(5) Sleuth--实现服务的链路追踪
